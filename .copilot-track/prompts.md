# Reusable Copilot Prompts — Crawl Track

## Plan First
> "Propose the smallest plan and diffs to meet THIS acceptance; list files, risks, and test plan; then show diffs file-by-file."

## Write Tests
> "Create an Arrange-Act-Assert test for <fn> with one valid and one invalid case; deterministic only."

## PR Draft
> "Fill this PR template for Exercise <id> using our changes; include commands, outputs (logs/tests), and a rollback note."

## Error Handling
> "From the last commit(s), propose 1 minimal error-handling improvement + a matching failure test."

## Time Guidance
- Average per exercise: 20–40 minutes
- Complex repos (first couple): up to 45–60 min
- Exercises 6–7 take longer; speed improves after
